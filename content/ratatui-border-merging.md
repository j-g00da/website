[Ratatui](https://github.com/ratatui/ratatui) v0.30 will introduce **block border merging**,
a feature that previously required manual handling.

You can now set the border merge strategy using `Block`â€™s new method: `merge_borders`

ğŸ“š **From the docs:**  

```rust
use ratatui::symbols::merge::MergeStrategy;

// Given several blocks with plain borders (1)
Block::bordered();
// and other blocks with thick borders (2) which are rendered on top of the first
Block::bordered()
    .border_type(BorderType::Thick)
    .merge_borders(MergeStrategy::Exact);
```
